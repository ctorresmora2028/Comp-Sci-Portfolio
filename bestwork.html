BEST WORK

This game is an example of what I think was my best work because I was able to create a game that put me through many difficulties, but in the end, I was able to make it come out really well, and I’m very proud of it. What made me feel proud of this project is that I was able to figure out alone how to get everything to work and fall the way I wanted it to, so the bunny could reach the falling objects. What I think makes this work stand out is that I feel like it’s a very creative idea, and it was a cute one for the time it was made, which was in April. I found this work to be somewhat easy, but that doesn’t mean I didn’t go through any troubles because I definitely did in certain moments. The skills and knowledge I used in order to create this work were my creativity and the ability to just trust the process and completely change things so quickly in case there was something I wanted to add.
